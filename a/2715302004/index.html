bulb switcher

题目

There are n bulbs that are initially off. You first turn on all the bulbs. Then, you turn off every second bulb. On the third round, you toggle every third bulb (turning on if it’s off or turning off if it’s on). For the i-th round, you toggle every i bulb. For the n-th round, you only toggle the last bulb. Find how many bulbs are on after n rounds.

分析

输入n个灯泡，第一次打开所有的灯泡，第二次每两个更改灯泡状态，第三次每三个更改灯泡状态…第n次每n个更改灯泡状态。求出n次之后，所有亮灯的灯泡个数。

我们假设n=5来看：

初始状态： X X X X X

第一次： √ √ √ √ √

第二次： √ X √ X √

第三次： √ X X X √

第四次： √ X X √ √

第五次： √ X X √ X

经过五次遍历之后，只有1号和4号是亮着的，他们是平方数。

对于第n个灯泡，当遍历次数是n的因子的时候，才能更改灯泡状态，即n能被当前遍历次数整除。比如n=36时，它的因子有(1,36)，(2,18),(3,12),(4,9),(6,6)，前四个括号对成对出现的次数抵消了灯泡变动，等于没有变化。只有在次数6的时候，才变了一次而且不能再变回去了。

比如第34个灯泡，因子有(1,34)，(2,17)。每个括号对中作用抵消，所以相当于没有变动。

因此这个题目就变成了，求1~n中间的完全平方数。

Bulb Switcher II

题目

  1. There is a room with n lights which are turned on initially and 4 buttons on the wall. After performing exactly munknown operations towards buttons, you need to return how many different kinds of status of the n lights could be.

    Suppose n lights are labeled as number [1, 2, 3 …, n], function of these 4 buttons are given below:

    1. Flip all the lights.
    2. Flip lights with even numbers.
    3. Flip lights with odd numbers.
    4. Flip lights with (3k + 1) numbers, k = 0, 1, 2, …

思路

上面一个题目的拓展，关灯的方式变了，有四种关灯的方法：全关，关偶数灯，关奇数灯，关3k+1的灯。现在给你n个灯，初始状态都是开着的，允许m步操作，问我们总共能组成多少种不同的状态。

直接记答案吧。。并没有看懂推导过程

这道题一共有8中情况：

1、当m和n其中有任意一个是0的时候，返回1。

2、当n=1时，只有两种情况，0或者1

3、当n=2时，这个时候要看m的次数

  • 如果m=1，那么有四种状态：00，01，10
  • 如果m>1，那么有四种状态：00，01，10，11

4、当m=1时，此时n>=3，因此有四种状态：000，010，101，011

5、当m=2时，此时n>=3，我们有七种状态：111，101，010，100，000，001，110

6、当m>2时，此时n>=3，我们有8种状态：111，101，010，100，000，001，110，011
