调整数组顺序使奇数在偶数前面

题目

输入一个数组，实现一个函数来调整该数组中数字的顺序，使得所有的奇数位于数组的前半部分，所有的偶数位于数组的后半部分。

思路

一个很自然的思路就是用两个指针，一个从前往后找偶数，一个从后往前找奇数。找到这样的一对之后，就将他们swap一下。

调整数组顺序使奇数在偶数前面Ⅱ

题目

输入一个整数数组，实现一个函数来调整该数组中数字的顺序，使得所有的奇数位于数组的前半部分，所有的偶数位于位于数组的后半部分，并保证奇数和奇数，偶数和偶数之间的相对位置不变。

思路

这里注意了，需要保持原来数字的顺序不变，也就是说要有稳定性。

所以我们就不能直接swap了，直接swap肯定是不能保证顺序的。我们的思路如下：

1、遍历数组，如果找到一个数字是奇数，那么就这个数字前面所有的偶数往后移，直到找到一个奇数为止；

2、将一个奇数后面为止替换成这个奇数。

Move Zeros

题目

Given an array nums, write a function to move all 0‘s to the end of it while maintaining the relative order of the non-zero elements.

Example:

Note:

  1. You must do this in-place without making a copy of the array.
  2. Minimize the total number of operations.

思路

给一个数组，要求把数组里所有的0移动到数组尾部，并且不改变非0元素的相对位置。

直接遍历一遍数组，如果元素非零，然后赋值给index位置，并且index++就可以了。
