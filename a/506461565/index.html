二叉树的高度和路径问题

本文字数： 4.2k 阅读时长 ≈ 4 分钟

二叉树的路径问题，分为两种情况，第一种是二叉树的高度问题，第二种是二叉树的路径。所谓高度问题就是从root到叶子节点，叫做高度。所谓路径就是任意两个节点，路径可以不通过root。

二叉树高度问题

求二叉树高度问题，是比较常见的一类问题。比如二叉树最大深度、最小深度、是否是平衡二叉树、任意两个节点之间的最长路径题目，都需要找二叉树的高度。这里总结一下跟二叉树高度有关的题目，试图总结出一个统一的方法。

二叉树路径问题

二叉树中任意两个节点的路径，一定要用DFS来考虑。

这几个题目都有着统一的思想：

  • 以通过当前root路径考虑，使用递归计算左子树、右子树的相应值。
  • 计算通过当前路径的满足条件的值
  • 更新全局变量
  • 返回当前root情况下的结果

总结为一类问题：二叉树中的路径问题。

4、Diameter of Binary Tree

任意两个节点的最长路径

题目

Given a binary tree, you need to compute the length of the diameter of the tree. The diameter of a binary tree is the length of the longest path between any two nodes in a tree. This path may or may not pass through the root.

Example:
Given a binary tree

Return 3, which is the length of the path [4,2,1,3] or [5,2,1,3].

思路

给一个二叉树，要求找任意的两个节点之间最长的路径。不同于之前的树深问题，这里可以不从root开始。但是我们可以这么考虑，对于每个节点p，我们去找左子树最深路径和右子树最深，然后两个加起来+1，就得到了当前这个节点p的最长路径了。

思路

因为是求路径长度，我们假设当前节点是root，那么可以理解为求与root具有相同值的左子树高度+右子树高度。这样问题久变成了求二叉树高度的变种问题。

  • 当前节点为root，计算左子树高度l=helper(root.left)，计算右子树高度r=helper(root.right)
  • 判断！如果root.left.val != root.val，那么l=0；root.right.val != root.val，那么r=0；
  • 计算通过当前root节点的值相同的路径长度len=l+r+1；
  • 更新maxlen=Math.max(maxlen, len)
  • 返回当前root节点的值相同的最大高度max(l,r)+1。
