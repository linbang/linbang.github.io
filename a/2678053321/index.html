思路

要求给一个字符串数字，判断它可以被解析成多少个字母组合。考虑使用dp，dp[i]表示以i位置数字结尾可以被解析成多少个字母组合。

对于第i位置的数字：

1、如果是0，那么dp[i]=0。

2、如果不是0：

​ 1、单独的看这个位置，他肯定可以被解析成一个字母。

​ 2、看当前位置和前一个位置组成的数字和26进行比较，如果比26大，那么这两个位置就不能组成字母，如果比26小，那么这两个位置就可以组成字母。

综上所述，我们可以写出状态转移方程：

dp[i] = 0 if s[i]==’0’

dp[i]=dp[i-1] if s[i-1,i] > ‘26’ else dp[i-1]+dp[i-2]
