Largest Rectangle in Histogram

题目

Given n non-negative integers representing the histogram’s bar height where the width of each bar is 1, find the area of largest rectangle in the histogram.


Above is a histogram where width of each bar is 1, given height = [2,1,5,6,2,3].


The largest rectangle is shown in the shaded area, which has area = 10 unit.

Example:

思路

有名的直方图问题。

思路一：双指针，时间复杂度$o(n^2)$，空间复杂度$o(n)$

1、遍历数组，找每一个局部峰值。所谓的局部峰值，就是当前的数字大于后面一个数字，就叫做局部峰值。

2、对每一个局部峰值，前向遍历找矩形的最大面积

为什么只需要计算局部峰值呢？对于非局部峰值，比如1，他后面的5比它大，那么一定是可以包含住着个小矩形的。所以不许用考虑非局部峰值的点。

思路二：DP，时间复杂度O(n)，时间复杂度O(n)

这个题题目，乍一看非常像containing water(双指针)、traping water（双向dp）、二次买卖股票(双向DP)、最大乘积数组（双向dp）这种类型的题目，所以考虑使用DP来做。

因为要计算最大的矩形面积，那么分解一下我们试图找到以每个位置height[i]高度的最宽的距离。

那么我们设置两个dp数组：

1、left[i]：表示>=height[i]的最左边的位置的前一个位置；right[i]：表示>=height[i]的最右边的位置的后一个位置。

2、left[i]=-1;right[i]=length;

3、状态转移：形象的描述，就是while循环不断的往单边方向找。

4、return ： max(height[i] * (right[i] - left[i] - 1), area )
